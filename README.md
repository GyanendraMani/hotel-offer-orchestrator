# Hotel Offer Orchestrator

## How to Run

### 1. Clone repository

```bash
git clone https://github.com/GyanendraMani/hotel-offer-orchestrator.git
cd hotel-offer-orchestrator
```

### 2. Run application

```bash
docker-compose up --build
```

### 3. Access API

```bash
http://localhost:3000/api/hotels?city=delhi
```
